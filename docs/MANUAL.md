# Modiq 운영 매뉴얼 (v1.2.0)

> 모델 에이전시 운영자·매니저를 위한 사용 안내서.
> 화면 좌측(모바일은 상단) 메뉴 순서대로 설명합니다.

---

## 0. 시작하기

1. **회원가입 → 에이전시 생성**: 첫 사용자는 회원가입 후 에이전시를 만들면 자동으로 **대표(owner)** 가 됩니다.
2. **회사 정보 입력**(회사정보 메뉴): 에이전시명, 대표 연락처(`rep_phone` — 알림톡 담당자 fallback), 사업자등록증(OCR 자동 인식) 등록.
3. **멤버 초대**(담당자 메뉴): 매니저를 추가하고 각자 **연락처(phone)** 를 채웁니다 → 알림톡 담당자 연락처로 사용.
4. **요금제 선택**(요금제 메뉴): Starter / Standard / Pro. 알림톡은 Standard 이상.

## 1. 대시보드

- 상단 통계 카드(진행중 섭외·이번 달 매출 등).
- **신규 문의 카드**: 상태가 `문의접수(INQUIRY)`인 건을 모아 보여줍니다. 오늘 들어온 건은 **NEW**, 묵은 건은 **N일 경과**(3일↑ 빨강).
  - 행 클릭 → 섭외 상세에서 **확정/취소**로 전환.
  - ※ 문의접수 건은 "진행중 섭외" 통계에서 제외(중복 방지).
- 이번 주 섭외, 날짜 클릭 시 캘린더 이동.

## 2. 모델 관리

### 2-1. 모델 등록 (모델 메뉴)
- 개별 등록: 이름·**생년월일 6자리(YYMMDD)**·성별·모델 타입·신체·활동분야·연락처.
- **외국인 모델**: 상단 "외국인 모델" 토글 → **[비자·정산 정보]** 모달에서 비자 유형·입출국일·세율·지급방식 + **주소·외국인등록번호/여권번호**까지 **한 화면에서** 입력(정산 세무유형 자동 반영).
- **세무 신고용 정보**(내국인 주민등록번호 / 외국인 식별번호)와 **세무 식별번호**는 **대표·정산권한자(재무 열람)** 계정에만 보입니다. 권한 없는 멤버에게는 숨겨지는 것이 정상입니다.
  - 식별번호는 **암호화 저장**되어 화면엔 마스킹만 표시됩니다. 세무 주민번호를 입력하면 위의 생년월일 6자리가 자동으로 채워집니다.
- **3사이즈(가슴·허리·엉덩이)**: inch 우선 입력(기본 inch). cm로 입력하면 inch로 자동 변환되어 표기됩니다(저장은 cm).
- **일괄 등록**: 엑셀 업로드(일괄 등록 버튼).

### 2-2. 포트폴리오 (포트폴리오 메뉴)
- 좌측에서 모델 검색·선택 → 우측에 **프로필 + 갤러리**.
- 사진은 드래그앤드롭 또는 ＋로 추가(**최대 30장**), 드래그로 순서 변경.
- **대표 지정**: 갤러리 사진의 "대표 지정" 또는 "대표 사진" 업로드 → 목록·카드 썸네일로 사용.
- **컴카드 만들기**: 사진 1장 이상이면 활성화. 슬롯(좌1 + 우 2×2)에 사진을 배치해 A4 가로 컴카드 생성 → 미리보기 후 다운로드/공유.
- **속도 팁**: 좌상단에 **"썸네일 생성"** 버튼이 보이면 1회 실행하세요. 기존 사진의 작은 썸네일을 만들어 목록·갤러리 로딩과 데이터 사용량을 크게 줄입니다(완료 후 버튼 사라짐). base64로 저장된 옛 사진은 **"저장소로 이전"** 버튼으로 옮기면 로딩이 빨라집니다.
- 사진은 저장 시 자동으로 **긴 변 1500px(품질 0.8)** 로 정리되고, 갤러리용 **480px** / 목록·아바타용 **150px** 썸네일이 만들어집니다.

### 2-3. 모델 검색
- 성별·국적·모델타입·활동분야·머리길이·나이·신장·신발·모델료 범위로 필터.
- ※ 값이 비어 있는 모델은 범위 필터에서 숨겨지지 않습니다(누락 방지).

## 3. 패키지 (모델 제안)

- 정보 완비 모델(이름+사진+키)을 선택 → **패키징** → 공유 링크 자동 생성·복사.
- 패키지 메뉴에서 제목·고객사 편집, 공개 페이지 확인, PDF 발송.
- 공개 페이지: 카드는 대표 1컷만, 클릭 시 갤러리. 모델 신체정보는 자동 반영(재입력 불필요).

## 4. 섭외 관리

### 4-1. 섭외 추가
- 타입 선택: **촬영 / 실물미팅 / 피팅 / 오디션**.
- 모델·고객사·담당자(멤버 드롭다운)·일시·장소·금액 입력.
- 카카오/채팅으로 들어온 문의는 상태=**문의접수**로 등록 → 대시보드 신규 문의 카드에 노출.

### 4-2. 상태 흐름
- 촬영: 문의접수 → 모델제안 → 선택대기 → 스케줄확인 → (HOLD) → 섭외확정 → 촬영완료 → 정산완료 / 취소.
- 비촬영(미팅·피팅·오디션): **요청 → 확정 → 완료**(+취소)로 단순화.

### 4-3. 휴무 충돌
- 모델 기간 휴무(`model_offs`)와 겹치는 일정은 추가·수정 시 **경고 후 강행 허용**.
- 캘린더에 모델별 음영 / "휴무 N" 표식.

### 4-4. 스케줄 충돌 / 자동 HOLD(보류)
- 같은 모델·같은 날 일정은 **먼저 시작하는 일정** 기준으로 다음 간격을 둡니다.
  - **미팅·피팅·오디션끼리 같은 장소** → 시간이 겹쳐도 **HOLD 안 함**.
  - **미팅·피팅·오디션끼리 다른 장소** → **1시간**(이동).
  - **앞=미팅류 → 뒤=촬영** → **1시간**. (예: 11~12시 미팅 → 13시 촬영 가능)
  - **앞=촬영 → 뒤(촬영·미팅 무관)** → **2시간**.
- 충돌 시 **촬영 우선** → 미팅류가 **HOLD**로 내려갑니다. 시간/장소를 충분히 벌리면 **자동으로 HOLD 해제**(→ CHECKING).
- 캘린더에서 충돌 건은 색상으로 강조됩니다(겹침/간격부족 구분).

## 5. 정산 · 매출

### 5-1. 정산 (정산 내역서)
- 필터: **날짜 / 모델 / 고객사**. 건별 리스트가 기본.
- 모델료(Day 9h / Half 5h / Hour 1h) + 정산방식(비율 % / 정액)으로 모델 정산액 산출.
- 세무유형: 외국인(비자율) / 프리랜서 3.3% / 소속사 +10% 계산서.
- **모델 발급 서류(단일 모델 선택 시)** — 세무유형으로 자동 분기:
  - **프리랜서·외국인 → 원천징수 내역서** (식별번호 라벨: 내국인=주민등록번호 / 외국인=여권번호·외국인등록번호)
  - **소속사(대대행) → 거래명세서** (공급가액+부가세 10%+합계). 소속사는 원천징수 대상이 아니며 **세금계산서는 소속 에이전시가 발행**합니다.
  - 미리보기 → 🖨 인쇄/PDF 저장 → 📧 이메일 발송.
- **세무사용(주민번호 포함)** 엑셀은 대표·정산권한자만 받을 수 있고, 다운로드 시 보안 감사 로그가 남습니다.

### 5-2. 매출 현황
- 순위에서 모델/고객사 선택 → 해당 매출만 필터(요약 카드 연동).
- **매출총이익 = 매출(공급가) − 모델 정산 기준액** 카드 표시.
- **엑셀(.xlsx) 다운로드**(섭외ID·계약금·잔금·모델지급·총이익·합계).
- 건별 내역에 고객사 **입금/미입금** 표시.

### 5-3. 매출 거래명세서 / 청구 (고객사 발급)
- 매출 화면 상단 **`📑 거래명세서 / 청구`** → 고객사 대상 거래명세서(공급자=당사) 발급.
- **발급 단위 토글**:
  - **프로젝트별**: 프로젝트 1개 = 명세서 1장(프로젝트명 없으면 단건).
  - **고객사 월합산**: 한 고객사의 기간 전체 거래를 **한 장**에 묶어 발급.
- **월별 리스트**: 고객사·프로젝트·공급가·청구(VAT)·미수금·상태(**완납/부분입금/미수**) + 상단 합계카드(총청구·입금완료·미수금) + `미수만 보기` 필터.
- 행의 **`🧾 명세서 발급`** → 미리보기(공급가액+부가세 10%+합계+입금현황) → 인쇄/PDF / 이메일(고객사 '계산서 발송 이메일' 우선).
- 입금 상태는 섭외 상세의 **계약금/잔금 입금 체크(또는 완납)** 를 기준으로 잡힙니다. ⚠️ 실제 세금계산서는 홈택스에서 별도 발행하세요.

## 6. 캘린더 동기화

### 6-1. 확정 → 모델 수락 → 동기화 (기본, v1.2.0)
- 촬영·실물미팅을 **확정**하면 모델에게 **[수락]/[거절] 초대 메일**이 자동 발송됩니다. (이 시점엔 캘린더에 일정이 만들어지지 않음)
- 모델이 메일의 **[수락]** 을 누르면 → 에이전시 구글 캘린더에 일정이 생성되고, 모델은 수락 페이지에서 **.ics·구글 추가**로 자기 캘린더에 넣습니다. (모델 메일이 구글이 아니어도 OK)
- **거절** 하면 담당자에게 알림 메일이 갑니다. 섭외 상세에 **수락 대기 / 수락됨 / 거절** 배지가 표시됩니다.
- 메일 발신자는 **modiq**, 모델이 **회신하면 우리(에이전시) 이메일**로 옵니다.
- ⚠️ 초대 메일이 실제 발송되려면 **Resend 도메인 인증 + 발신주소(EMAIL_FROM)** 설정이 되어 있어야 합니다.
- 수락을 기다리지 않고 **즉시** 넣고 싶으면 섭외 상세 **"일정 보내기 → 구글 캘린더 등록"**(수동 override)을 쓰세요. "모델 이메일"·"링크" 옵션도 그대로 있습니다.

### 6-2. 구독형 (자동 동기화) — 권장
- 모델 상세 → **"구독 링크"** 버튼으로 안내 페이지 링크(`?sub=...`)를 복사해 모델에게 전달(카톡/메일 무관).
- 모델이 링크를 열면 기종 자동 감지:
  - **아이폰**: "캘린더에 구독 추가" 탭 1회.
  - **안드로이드**: ① Play 스토어에서 **ICSx⁵** 설치 → ② "ICSx⁵로 구독 추가". (구글 캘린더 앱은 URL 구독 직접 불가)
  - **PC**: 구글 캘린더 → 다른 캘린더 ＋ → "URL로 추가".
- 구독은 **최초 1회**. 이후 확정 일정 변경·취소가 자동 반영.
- ⚠️ 동기화는 "주기적"(실시간 아님). **급한 변경·취소는 알림톡/이메일로 별도 통보**, 구독 피드는 전체 일정 자동 유지용.

## 7. 카카오 알림톡 (Standard 이상)

- 4종: **확정 / 취소 / 전날 리마인드 / 변경**. 발신번호·템플릿은 카카오 사전 승인 필요.
- 담당자 연락처 우선순위: 멤버 `phone` → 에이전시 `rep_phone` → 없음.
- 외부 준비: 카카오 비즈니스 채널 개설 → 템플릿 4종 승인 → `solapi-send` 배포 → `VITE_SOLAPI_FN_URL` 설정. 준비 전에는 발송이 no-op로 안전.

## 8. 권한 · 요금제

| 권한 | 설명 |
|---|---|
| **대표(owner)** | 설정·담당자 관리·재무 노출 전권. 공동 대표 권한 부여 가능. |
| **멤버(member)** | 일반 운영. 재무 열람은 별도 부여 시에만. |

### 8-1. 대표 변경 / 권한 이전 (소유권)
대표 권한은 **각 담당자의 역할(role)** 로 정해지며 **옮길 수 있습니다**(공동 대표 가능). 위치: **설정 → "권한 부여 (공동 대표)" 카드**.

- **권한 부여(공동 대표)**: 담당자 선택 → `권한 부여` → 그 담당자도 대표(owner)가 됩니다. (본인 권한 유지)
- **권한 회수**: 같은 카드 아래 **"현재 대표 권한 보유자"** 에서 이름 옆 **✕** → 일반 담당자로 내림.
  - ⚠️ **본인 권한은 회수 불가** (자기 옆엔 "(나)"만 표시).
  - ⚠️ **대표는 최소 1명 유지** (마지막 대표는 회수 불가 — 계정 잠김 방지).

**대표를 완전히 교체하는 순서**
1. (기존 대표 로그인) 새 사람을 **담당자 추가** → 설정에서 그 담당자에게 **권한 부여**(공동 대표).
2. **로그아웃 → 새 대표 계정으로 로그인.**
3. 새 대표: 설정 → "현재 대표 권한 보유자" → **기존 대표 옆 ✕** 로 권한 회수.
4. (선택) 담당자 관리에서, 일반 담당자가 된 기존 대표를 **삭제**.

> 대표(owner)는 담당자 화면에서 바로 삭제되지 않습니다 — **권한 회수로 일반 담당자로 내린 뒤** 삭제 버튼이 생깁니다.
> ⚠️ 기존 대표가 **누구에게도 권한을 안 주고 떠나면** 대표 기능이 잠깁니다(아무도 owner 아님) → DB에서 다른 담당자를 owner로 승격해야 복구.
> 참고: 로그인 이메일과 `owner_email`(모델 메일 **회신주소**)은 최초 생성자 기준으로 남으니, 대표가 바뀌면 회신주소도 함께 변경을 권장합니다.
